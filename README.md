# Incomprobable UI

## Installation

```bash
# Clone Repository
git clone https://github.com/Ex-tendeal/incomprobable-ui

# Move to folder
cd incomprobable-ui

# Install library dependencies
yarn install
```

## React Native App

```bash
# Clone Repository
cd stories/native

# Install library dependencies
yarn install
```

----

## Build

### Build for Development

```bash
# Build web bundle
yarn build:dev:web

# Build native bundle
yarn build:dev:native
```

### Build for Production

```bash
# Build web bundle
yarn build:prod:web

# Build native bundle
yarn build:prod:native
```

----

## Running in Local

### Running Web in Local

```bash
# Run in web
yarn storybook:web

# Open Storybook
Navigate to http://localhost:6006
```

### Running Native in Local

```bash
# Run in web
yarn storybook:native

# Open Storybook
Navigate to http://localhost:6006
```
